Add a Language Wizard

The Add a Language wizard can be used to create additional translation files in order to localize the user interface. It can be started from the The NLS Editor.

It is possible to crate one translation file that corresponds to the user locale, which are specified by a language and optionally a country (de, de_DE, de_CH and so on).

Language

The language of the locale.

Country

(optional) The country of the locale.

Folder

The location where the Texts properties file will be created.
