New Scout Project Wizard

The New Scout Project wizard can be used to create a new Scout project. To open it, select the File ▸ New ▸ Project…​ Menu and type “Scout Project” in the wizard search field.

Group Id

Maven groupId used for all created projects.

Artifact Id

Maven artifactId for the parent project. The additional projects are derived from this name (If you select myapp you will get several other projects myapp.client, myapp.shared, …​)

Display Name

The name of the application presented to the user (e.g. in the Browser title bar).

With the Project Location group box, you can control where the project will be created. Unchecked the Use default Workspace location checkbox to enter an other value in the Target Directory Field. The Browse…​ button can help you to find the appropriate path.

See also the Hello world Tutorial for a complete project creation example.
